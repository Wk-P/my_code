`problem3_ppo_opt` 是 `problem3_ppo` 的**实验性副本**，共享相同的环境设计和算法。

## MDP 建模（摘要）

**目标：** 最大化 AR = $\frac{1}{M}\sum x_{ij}\frac{n_i}{e_j}$。

**状态：** $s_t = [d_t,\; AR_t,\; r_0^t, \ldots, r_{N-1}^t]$ — 形状 $(N+2)$。

**动作：** $a_t \in \{0, \ldots, N-1\}$ — 为当前服务选择 ECU。

**奖励：**
$$r_t = \begin{cases} AR_M & t = M-1 \\ 0 & \text{其他} \end{cases}$$

无违反惩罚，违反情况仅记录在 `info` 中。

## 算法与超参数

**算法：** PPO（Stable-Baselines3）

| 超参数        | 值           |
| ------------- | ------------ |
| 学习率        | `3e-4`       |
| `n_steps`     | `1024`       |
| 批次大小      | `128`        |
| 更新轮数      | `10`         |
| $\gamma$      | `0.999`      |
| GAE $\lambda$ | `0.95`       |
| 裁剪范围      | `0.2`        |
| 网络结构      | `[256, 256]` |
| 总步数        | `200,000`    |
